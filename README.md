##  Features

###  Core Functionality
-   Browse through a curated collection of books across multiple categories
-  Real-time search with filtering by title, author, and category
-  Add, remove, and manage items with quantity controls
-  Save favorite books for later
-  Optimized for desktop, tablet, and mobile devices

-  Loading shimmers, hover effects, and micro-interactions
-  Toggle between light and dark themes
